# ⚡ Sistema Web de Orçamentos — Manutenção de Quadros Elétricos

Aplicação web desenvolvida para substituir um processo manual de orçamentação em uma empresa do setor elétrico. Permitindo maior controle, agilidade e confiabilidade.

---

## Funcionalidades

* Múltiplos usuários operando simultaneamente sem conflito
* Cálculo de custos em tempo real no frontend
* Dashboard com gráficos e controle de status dos orçamentos
* Exportação de relatórios em Excel
* Gestão completa de orçamentos

---

## Tecnologias

### Backend

* ASP.NET Core
* Entity Framework Core

### Frontend

* React

### Banco de Dados

* PostgreSQL

### Outros

* ClosedXML (geração de Excel)
* IIS (deploy em Windows Server)

---

## Problema Resolvido

Antes:

* Processo manual
* Alto risco de erro
* Baixa produtividade

Depois:

* Automação completa / Redução de erros
* Fácil compartilhamento
* Maior velocidade na geração de orçamentos
